Project Overview

Work Myself is a mobile application designed to help users build and maintain a personalized workout routine. The app allows users to add exercises for each day, which are then displayed on a clean and intuitive dashboard. Its goal is to make daily fitness tracking more structured, engaging, and motivating.

Beyond workouts, the app also provides a journal-like notes feature where users can log their thoughts and progress. Additionally, by tapping the clock on the dashboard, users receive motivational quotes to inspire consistency and positivity throughout their fitness journey. The app uses SQLite for reliable local storage and smooth offline functionality.

Key Features

Daily Workout Routines

Users can create and manage custom workouts that are displayed directly on the dashboard.

Built-in Journal

Log notes and track progress with an integrated journal system to support fitness goals.

Motivational Quotes

Receive inspirational quotes by interacting with the dashboard clock for daily encouragement.

SQLite Database

Secure and reliable local storage ensures workouts, notes, and data are always accessible offline.
